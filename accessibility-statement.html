Accessibility statement

Our commitment to an accessible, usable website for everyone.

Last updated: July 24, 2026.

Our commitment

We want everyone to be able to browse our catalog and place an order. We work toward conformance with the Web Content Accessibility Guidelines (WCAG) 2.2 Level AA.

What we’ve done

Semantic HTML structure and landmarks; a skip-to-content link; descriptive alt text on product imagery; labelled form fields with autocomplete; visible keyboard focus indicators; and color choices intended to meet AA contrast for body text.

Ongoing work

Accessibility is continuous. We test with keyboard navigation and screen readers and fix issues as we find them.

Report a problem

If you run into a barrier on this site, email info@stackedbiologics.com with the page and a short description, and we’ll address it as quickly as we can — usually within one business day.
